# eslint-config-centra

A shareable ESLint configuration used in Centra projects.

## Usage

1. Install the package.

    ```bash
    npm i -D github:centrahq/eslint-config-centra
    ```

1. Add `eslint-centra-config` to your `.eslintrc`.

    ```json
    {
      "extends": [
        "centra"
      ]
    }
    ```
